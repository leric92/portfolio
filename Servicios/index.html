En la carrera hacia la digitalización las páginas web y aplicaciones se vuelven parte fundamental de cualquier empresa. Una página web bien definida puede ser la diferencia entre ganar o perder un cliente.
En muchos de los casos dado la premura de los cambios diversas empresas crean páginas web cuyo impacto es nulo tanto para los internautas de la web como para los propios buscadores

El peor error al usar redes sociales para una empresa es pensar que puede ser tratada como una red social personal. Ya que si bien su uso es similar el fin es totalmente diferente.
Una red social personal es un espacio único que se caracteriza por elementos distintivos propios de su creador. Una red social empresarial es un espacio universal que se caracteriza por las forma en como se dan las interacciones dentro de este grupo precedidas por sus miembros

De la misma forma en como creamos experiencias de servicios o alrededor de los productos. Los sitios web deben crear experiencias para todos aquellos usuarios que entren al mismo. Un sitio web que crea u ofrece una experiencia para el usuario, es un sitio web que lograra diferenciarse de los demás sitios en la web.
Las experiencias pueden ir más allá de elementos técnicos como usabilidad, pueden partir del uso de elementos visuales capaces de apelar a los sentidos, mente e imaginación.
